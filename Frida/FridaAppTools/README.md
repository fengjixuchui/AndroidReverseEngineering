## FridaHooker：

    Android平台可视化Frida管理工具，图形界面版本的frida，提供一个方便地管理frida的方式，告别命令行。

    一、主要功能
    
        从本地安装frida server到root后的手机
        启动、关闭frida server
        
    二、使用方法
    
        打开应用，然后安装frida server后，打开开关启动frida即可。
        
    三、环境配置
    
        系统要求：Android 5.0或更高版本，需要应用具有root权限。
